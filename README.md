# 📈 Stock Price Management System

A comprehensive web application for managing stock portfolios with real-time price fetching from multiple market APIs.

![Stock Management System](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🌟 Features

- **📊 Stock Management**: Add, update, and delete stocks with ease
- **🔍 Search Functionality**: Find stocks by company name (partial matching)
- **🌐 Live Market Data**: Fetch real-time stock prices from multiple sources
- **🔄 API Integration**: Supports Alpha Vantage and Yahoo Finance
- **💾 Local Storage**: Save API keys and preferences
- **📱 Responsive Design**: Works on desktop and mobile devices
- **⚡ Fallback System**: Always returns realistic prices even when APIs fail
